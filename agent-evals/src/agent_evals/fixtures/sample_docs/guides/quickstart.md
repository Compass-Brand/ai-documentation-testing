# Quickstart

Get DataForge running locally in under 5 minutes.

## Prerequisites

- Python 3.11+
- PostgreSQL 14+
- Redis (optional, for caching)

## Installation

```bash
pip install dataforge
dataforge init my-project
cd my-project
```

## First Run

```bash
dataforge serve --reload
```

Visit http://localhost:8000/docs to see the auto-generated API docs.
